# **calculator 计算器**


_江苏大学物联网工程专业C语言编程课程的学校实训作业。_

基于 C11 语言的轻量计算器应用程序，可以执行基本的算术运算。

## 特性

- 基本算术运算：加、减、乘、除。
- 基本运算：清除、删除、相等。
- 高级运算：平方根、立方根、阶乘等。
- 支持浮点数。
- 支持键盘输入。
- 支持鼠标输入。
